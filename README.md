# 自分用競プロライブラリ
[verification-helper](https://github.com/online-judge-tools/verification-helper/blob/master/README.ja.md)を使って、書き散らかしていたライブラリ整備を試みる。  

### インストール
```
$ pip3 install online-judge-verify-helper
```

### 実行
```
$ oj-verify run
```

### include展開
```
$ oj-bundle main.cpp
```

### ドキュメント生成
```
$ oj-verify docs
```
